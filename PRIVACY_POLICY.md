# Privacy Policy for DevLens Ultra

**Effective Date:** March 28, 2026

At DevLens Ultra, we are committed to protecting your privacy. This Privacy Policy explains how our Chrome Extension handles information.

## 1. Information We Collect
DevLens Ultra is a developer tool. It collects the following types of information only when actively triggered by the user:
-   **User Activity**: We capture mouse clicks and hover states to allow you to select DOM elements for analysis.
-   **Website Content**: We read the DOM structure and CSS properties of the active tab to generate code snippets. 

## 2. How We Use Information
The collected information is used solely to:
-   Provide the "Select Element" and "Code Generation" features.
-   Generate React JSX and Tailwind CSS snippets.
-   Provide AI-powered explanations if you explicitly use the AI features.

## 3. Data Transmission
-   **Local Processing**: Most analysis happens locally in your browser.
-   **AI Features**: If you use AI refinement features, the selected DOM snippet is sent to your configured AI provider (e.g., OpenAI, Groq) to generate a response. This data is not stored by DevLens Ultra after the session.
-   **No Third-Party Tracking**: We do not sell or share your data with third-party advertisers or trackers.

## 4. User Control
-   The extension only runs when you click the extension icon or use the "Select Element" tool.
-   You can manage or delete your stored API keys at any time through the extension's settings.

## 5. Contact
If you have any questions about this Privacy Policy, please contact us via our GitHub repository: [https://github.com/Arsh-pixel-cmd/DevLens](https://github.com/Arsh-pixel-cmd/DevLens)
